# Snake Game

A console-based Snake game for Linux using ANSI characters and colors.

## Features

- Classic Snake gameplay with wrap-around boundaries
- Configurable grid size and speed
- Colorful ANSI output
- Multiple control schemes (arrow keys, vim keys, ESDF)
- Pause, restart, and exit functionality
- Optional wall display (purely visual)

## Requirements

- Python 3.x
- Linux environment
- Terminal with ANSI color support

## Installation

No installation required. Simply run the game from the `snake` directory.
